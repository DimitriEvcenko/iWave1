Conforms to UICollectionViewDelegate
Declared in WLMainCollectionViewController.h
WLMainCollectionViewController.m

Overview

The main collection view comtroller handles navigation through iWaves mini apps. Additional it is the start view controller for the tutroial. That tutorial can work without failure, the view controller should only be left with segue.
