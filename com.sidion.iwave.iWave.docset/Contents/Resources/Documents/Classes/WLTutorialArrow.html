WLTutorialArrow.m

Overview

The singleton instance of this class contains a text view and a blinking arrow. Arrow and textfield can be shown seperatly but they are global and independent from any view controller. So they use global coordinates. To place the control elements in any orentation transformation methods transform the coordinate systems so that the origin is always on the upper left corner. Arrows angel is also transformatted so that a direction to the left is zero degrees and the degrees increase clockwise.
