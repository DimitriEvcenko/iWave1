WLVacationListViewController.m

Overview

The vacation list view controller is used as master in a split view with WLVacationDetailViewController. It contains a table view that shows all vacation demands managed by a fetched controller. So all changes in core data are shown in the table. The vacation demands are sorted to sections by theire demand status.
