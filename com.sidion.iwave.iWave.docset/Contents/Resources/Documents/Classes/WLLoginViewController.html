Overview

This class handles the Login-UI.

On this View the user can put his logindata to login to the wave-intranet.

There are two controls to get the users logininput (userIdInput, passwordInput). To save the logininput in core data the rememberMeSwitch can be used. To confirm the login action, loginButton is used. Errors are shown on the errorMsgLabel.
