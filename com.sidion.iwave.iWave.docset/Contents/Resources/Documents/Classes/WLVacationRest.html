Overview

This class handles communication with wave intranet server. The interface that is used are REST-Services with RestKit-Framework. The class only handles communication in vacation context.

demandStatusFromDisplayName:

Returns a special vacation demand type for reports, if user is manager.

- (NSString *)demandStatusFromDisplayName:(NSString *)displayName

Parameters

displayName

Displayed name of the report.

Return Value

The string to display reports vacation.

Discussion

Returns a special vacation demand type for reports, if user is manager.

Managers reports vacation demands that server asks for are normaly pending. To seperate them from the logged in users pending request, this method generates demand types for:

diectReports "TEAMVACATION";
othrReports = "OTHERREPORTSVACATION";

Warning: Userid cannot be used as primary key here, because vacation demand list does not return reports userIds.
