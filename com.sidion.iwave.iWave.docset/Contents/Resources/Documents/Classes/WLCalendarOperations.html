Inherits from NSObject
Declared in WLCalendarOperations.h
WLCalendarOperations.m

Overview

This class contains a lot of class methods that handle dates. The first task is to test a date if it is vacation date or what kind of vacation date it is. The second task is the comparison of dates like if dates are the same day or getting day difference of dates. The third task handles formatting the date to string. The fourth task is testing a date on public holiday or weekend.
