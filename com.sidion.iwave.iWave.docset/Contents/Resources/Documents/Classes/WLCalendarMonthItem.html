Overview

The class defines the model of a calendar month.

An Instance of this class can be displayed in the applications calendar view.

initWithStartDate:andEndDate:

Setting the month item to a real month.

- (void)initWithStartDate:(NSDate *)startDate andEndDate:(NSDate *)endDate

Parameters

startDate

Has to be the first day of the month.

endDate

Has to be the last day of the same month.

Discussion

Setting the month item to a real month.

The dayItems created are not only the days of the month. There are also days of the previous or following month to fill up weeks. Additional there are created day items to describe the weekdays a first.

Warning: Not yet testet, what happens, when start and end date are not correct!
