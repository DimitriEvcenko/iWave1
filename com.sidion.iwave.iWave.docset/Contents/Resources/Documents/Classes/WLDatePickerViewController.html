Overview

The date picker view controller is made to select a single date with day, month and year. The view controller can delegate changes in datePicker to a parent controller.

dateType

The date type defines if the view controller is used to set a start date or an end date of a vacation demand. So there are two types:

@property WLDateType dateType

Discussion

The date type defines if the view controller is used to set a start date or an end date of a vacation demand. So there are two types:

- START_DATE
- END_DATE
