Overview

This class handles the local vacation demands in core data.

Local vacation demands are stored as long as the demandStatus ist “unrequested”. This is necessary to seperate local persisted vacation demands from the session vacation demands. Furthermore the class is identical to VacationDemand.

Warning: This class is only used to store data, to display vacation demands VacationDemand is used.
