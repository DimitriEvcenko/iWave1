Overview

This class helps to map the different API models to a format that makes communication with wave intranet possible. Both plattforms are using Key-Value pairs to send data. So NSDictionary is the format to send and to recieve data. There are two big parts in this class:

1. Mapping received JSON-Strings to NSDictionary
2. Mapping any Request to NSDictionary

The third part: Mapping dictionarys to Objects is real small because of difficulites with creating NSManagedObjects from dictionary
