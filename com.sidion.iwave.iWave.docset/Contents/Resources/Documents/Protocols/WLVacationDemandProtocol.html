WLVacationDemandProtocol Protocol Reference

Conforms to NSObject
Declared in WLVacationDemandProtocol.h

Overview

This protocol defines the methods that are used on creating or editing vacation demands. In calendar view and vacation detail view modal popover views are used. To delegate changes to the parent views this protocol is created.
